# OWN-EC-FirstTestCheck
## 基于MPU6050和PID的尝试项目

---

### 引脚 + 定时器分配
#### [TIM定时器]
- `TIM1`:定时器定时中断
- `TIM2`:PWM波生成
- `TIM3`:编码器模式计数（正交解码）
- `TIM4`:编码器模式计数（正交解码）

#### [电机驱动]
- `PB12`:(TB6612)AIN1	一号电机方向 1
- `PB13`:(TB6612)AIN2	一号电机方向 2
- `PA1`:(TB6612)PWMA    一号电机速度

- `PB10`:(TB6612)BIN1	二号电机方向 1
- `PB11`:(TB6612)BIN2	二号电机方向 2
- `PA2`:(TB6612)PWMB    二号电机速度

#### [电机]
- `PB6`:一号电机的A相(绿)
- `PB7`:一号电机的B相(黄)
- `一号电机的M1`:(TB6612)AO2
- `一号电机的M2`:(TB6612)AO1

- `PA6`:二号电机的A相(绿)
- `PA7`:二号电机的B相(黄)
- `二号电机的M1`:(TB6612)BO1
- `二号电机的M2`:(TB6612)BO2

#### [蓝牙模块]
- `PA9`:串口模块RX
- `PA10`:串口模块TX
