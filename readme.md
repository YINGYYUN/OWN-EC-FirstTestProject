# OWN-EC-FirstTestCheck
## 基于MPU6050和PID的尝试项目

---

### 引脚 + 定时器分配
#### [TIM定时器]
- `TIM1`:定时器定时中断
- `TIM2`:PWM波生成
- `TIM3`:编码器模式计数（正交解码）
- `TIM4`:编码器模式计数（正交解码）

#### [电机驱动]
- `PB12`:(TB6612)AIN1	一号电机方向 1
- `PB13`:(TB6612)AIN2	一号电机方向 2
- `PA1 `:(TB6612)PWMA   一号电机速度

- `PB14`:(TB6612)BIN1	二号电机方向 1
- `PB15`:(TB6612)BIN2	二号电机方向 2
- `PA0 `:(TB6612)PWMB   二号电机速度

#### [电机]
- `一号电机的A相(绿)`:PB6
- `一号电机的B相(黄)`:PB7
- `一号电机的M1(红)`:(TB6612)AO2
- `一号电机的M2(白)`:(TB6612)AO1
- `一号电机的VCC(蓝)`:3.3V / 5V
- `一号电机的GND(黑)`:GND

- `二号电机的B相(黄)`:PA6
- `二号电机的A相(绿)`:PA7
- `二号电机的M1(红)`:(TB6612)BO1
- `二号电机的M2(白)`:(TB6612)BO2
- `二号电机的VCC(蓝)`:3.3V / 5V
- `二号电机的GND(黑)`:GND

#### [蓝牙模块]
- `PA9 `:蓝牙模块RX
- `PA10`:蓝牙模块TX
